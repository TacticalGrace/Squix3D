# Squix3D

A VR game for Meta Quest 3, built with Godot 4.6 and OpenXR.

## Requirements

- Godot 4.6+
- JDK 17
- Android SDK + Platform Tools
- Meta Quest 3 with Developer Mode enabled

## Godot Plugins

- [OpenXR Vendors Plugin](https://github.com/GodotVR/godot_openxr_vendors)
- [Godot Meta Toolkit](https://github.com/m4gr3d/Godot-Meta-Toolkit)
- [Godot XR Tools](https://github.com/GodotVR/godot-xr-tools)
